Vamos a crear una página web desde un repositorio de github

Teniendo nuestro repo ya creado con el archivo README.md hacemos una copia del archivo y lo nombramos index.md.

Si lo hacemos desde nuestro PC actualizamos el repositorio, nos dirigimos a él y nos vamos a Settings.

En Settings dejamos el nombre por defecto y bajamos a bajo del todo donde esta GitHub Pages:

Por defecto sale la rama None y nosotras la cambiaremos a master:

Ponemos la rama en master y le damos a Save ‘guardar’.

Esperamos un momento a que se genere el archivo y podemos ir a la página que nos muestra, como vemos en la imagen:

Como podemos ver se ha generado una dirección web y si entramos en ella veremos nuestro archivo README.md visualizado como una pagina web.
